⚙️ Pengaturan Koneksi API Database

Hubungkan UI Frontend Anda di GitHub ini dengan Database Google Sheets Anda.

Pastikan Anda telah menyetel deployment dengan akses "Anyone" pada Google Apps Script.

📖 Panduan Singkat Integrasi Google Sheets oleh Divisi Ubudiyah:

  1. Buka Google Spreadsheet baru Anda.
  2. Buat tab Mahasantri dan tab Absensi.
  3. Masuk ke Extensions > Apps Script, tempel kode Kode.gs.
  4. Klik tombol Deploy > New Deployment.
  5. Pilih tipe Web App, ubah Execute As ke Me dan Who Has Access ke Anyone.
  6. Salin link Web App yang Anda dapatkan lalu tempelkan pada input URL di atas.
